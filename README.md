# ATM Simulation - Exception Handling Project

## 📌 Description
This project is a robust ATM simulation written in Python. It focuses on preventing program crashes through effective exception handling and input validation.

## 🛠️ Programming Language
* Python 3.x

## 🛡️ Exception Handling Features
1. **ValueError:** Handles non-numeric inputs for withdrawal amounts.
2. **InsufficientFundsError:** A custom exception to handle business logic failures.
3. **General Exception Catch:** Ensures the program never crashes from unexpected runtime errors.

## 🚀 How to Run
1. Navigate to the `source_code` directory.
2. Run the command: `python atm_system.py`
3. Follow the menu prompts to interact with the system.
